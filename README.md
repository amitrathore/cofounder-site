# Cofounder

> Entrepreneurship is natural. AI just made it ours again.

The homepage for **Cofounder** — a book and platform by [Amit Rathore](https://github.com/amitrathore) on building anything, with anyone, in the age of AI and agents.

🌐 **Live site:** [cofounder.community](https://cofounder.community)

## About

*Cofounder: How to Build Anything, with Anyone* is a field guide for a new era of entrepreneurship — where a single person with an idea and a few agents can do what used to take a team of twenty.

The platform has four pillars:

1. **The Book** — the manifesto and the method
2. **The Community** — peers, mentors, and rooms to think out loud in
3. **The Exchange** — where founders find collaborators, capital, and customers
4. **The Field Guide** — a living, opinionated reference, free and updated continuously

## This repo

Static homepage. One file: `index.html`. No build step, no framework, no tracking. Open it in a browser.

### Local preview

```bash
# any static server works
python3 -m http.server 8000
# then open http://localhost:8000
```

### Deploy

The site is plain HTML/CSS and can be hosted anywhere — GitHub Pages, Cloudflare Pages, Netlify, Vercel, or any S3 bucket.
